# Belajar-Dasar-Pemrograman-Web

<ol>

  Pengenenalan <b>Website</b>.
  Pada pembelajaran ini saya memeplajari apa itu <b>Website</b>, sejarah <b>Website</b>, jargon teknis, struktur <b>Website</b>, dan anatomi <b>Website</b>.
